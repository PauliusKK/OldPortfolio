About Me

I’m a freelance Front End Web Developer based in Vilnius, Lithuania. I’m sure you just want to check out my work, so I’ll be quick. The work I provide is of high quality, fully responsive and tested in many browsers and devices. I ensure you that each project will be well-documentated and easily maintainable, so you’ll pretty much know what those blocks of code do.

Skills

I create responsive websites that allow the user to have the best experience in any device they’re using. Writing semantic markup that is documented and easy to maintain means if it scales in the future, it will be easy to maintain. I always use Prepros application to convert and compress the files I’m working on, that way the size of the files is decreased and website loads faster.

Clean (as well as organized) code is really important for me while working on the project. To achieve it I’m using preprocessors for HTML and CSS languages. That way I can convert many files into one by importing one file to another and so on, so forth.

I’m using Git on basically every project I’m working on. I love version control and couldn’t work without it. It keeps me sure that if something went wrong I still have a plan B. I can quickly and efficiently join your team using GitHub, Bitbucket or any other version control project.

Deadline is deadline and it’s my number one priority. I will do everything to deliver on time, no matter what. I respect every person and his time.
